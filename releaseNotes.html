v1.0.0 January 27, 2026

The Genesis Release

This milestone marks the stable release of ComponentMelt, introducing a new paradigm for building React modules that exist independently of their host applications.

  • Host-Agnostic Core: Strict boundary layer to prevent host app state pollution.
  • Externalized Reducer Pattern: Logic resides outside the module for maximum cross-project reusability.
  • Structural Integrity: Automated validation of required actions and reducers upon mounting.
